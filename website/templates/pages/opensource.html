Open Source

Join the Infinit open source community to contribute to the future of file storage.

Infinit is composed of several software layers from a C++ low-level asynchronous framework called ‘reactor’ to large-scale peer-to-peer networking and storage layers up to the file system.

Projects

We will be open-sourcing our projects over time to make sure every project is in a good state for the community to take over them. Feel free to come back or subscribe to our mailing-list to get notified.
